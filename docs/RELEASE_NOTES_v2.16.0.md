# ARNet Discovery v2.16.0

ARNet Discovery v2.16.0 provides a cleaner public Windows package, a refined product page, and documentation that helps engineers download, run, and evaluate the app with less guesswork.

## What is included

- Portable Windows x64 package for ARNet Discovery.
- Single-EXE publish configuration where supported by .NET/WPF publishing.
- Quick Start PDF and User Manual PDF included in the release ZIP.
- Product landing page with screenshot preview, clear download path, and engineering-focused FAQ.
- GitHub Actions workflows for CI, Pages deployment, and release packaging.

## How to run

1. Download `ARNetDiscovery-v2.16.0-win-x64-portable.zip` from the release assets.
2. Extract the ZIP to a local folder.
3. Run `ARNetDiscovery.exe`.
4. Select the Ethernet adapter connected to the engineering LAN.
5. Use **Scan Local**, **Probe**, or **Import Excel** + **Scan List**.

No installer, subscription, license key, or Visual Studio installation is required for normal use.

## Useful for

- FAT and SAT device visibility checks.
- Relay, IED, gateway, server, workstation, switch, meter, and printer IP verification.
- First-pass troubleshooting when expected devices do not appear on the engineering LAN.
- Exporting CSV evidence for project follow-up.

## Known limitations

- Open ports and protocol labels are engineering evidence, not formal device identity proof.
- Network policy, VLANs, routing, firewall rules, and blocked ICMP can affect scan results.
- Formal commissioning, protection testing, and cyber-security procedures remain the final authority for project acceptance.
