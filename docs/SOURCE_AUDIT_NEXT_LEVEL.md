# ARNet Discovery product roadmap and engineering notes

This page has been kept for existing links. For the current user-facing roadmap, see:

- [Product roadmap](ROADMAP.md)
- [User guide](USER_GUIDE.md)
- [Architecture](ARCHITECTURE.md)

ARNet Discovery is moving toward a simple product goal: provide a lightweight Windows tool that helps engineers verify expected devices against actual LAN and protocol evidence.

The next product improvements are:

1. safer protocol evidence validators;
2. project profile save/open support;
3. HTML evidence reports;
4. broader automated tests for importer, subnet, and classification logic;
5. cleaner contribution examples for real commissioning workflows.
